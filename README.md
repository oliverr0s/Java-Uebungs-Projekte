# Java Übungsprojekte

Dieses Repository enthält eine Sammlung meiner **Übungsprojekte in der Programmiersprache Java**.
Die Projekte dienen dazu, meine Kenntnisse in **objektorientierter Programmierung (OOP)** und beim Probleme lösen zu verbessern.

---

##  Inhalt des Repositories

Die Projekte sind in **separate Ordner** unterteilt.
Jeder Projektordner enthält:

* den vollständigen **Quellcode** 
* eine **kurze Beschreibung des Projekts**
* gegebenenfalls **Hinweise zur Ausführung**

---

##  Ziel dieses Repositories

Dieses Repository dient als **Portfolio**, um meine Fortschritte und praktischen Erfahrungen in der **Java-Entwicklung** darzustellen.

---


##  Projekte

Die einzelnen Projekte befinden sich in separaten Ordnern.
In jedem Ordner gibt es eine eigene **README-Datei**, die das jeweilige Projekt genauer erklärt.
